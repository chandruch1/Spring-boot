# 🔐 JWT Authentication Frontend

A modern and responsive frontend application built using **HTML, CSS, and JavaScript** that integrates with a **Spring Boot JWT Authentication Backend**. The application enables users to securely register, log in, and access protected resources using **JSON Web Token (JWT)** authentication.

This project demonstrates client-side authentication, secure token management, and REST API integration following modern web development practices.

# 📥 Installation

## Clone the Repository

```bash
git clone https://github.com/your-username/jwt-auth-frontend.git
```

## Navigate to the Project

```bash
cd jwt-auth-frontend
```

---

# ▶️ Run the Application

Open the project directly in your browser:

```text
index.html
```

or use **VS Code Live Server**.

### Using VS Code

1. Open the project.
2. Install the Live Server extension.
3. Right-click `index.html`.
4. Select **Open with Live Server**.

The application will be available at:

```
http://127.0.0.1:5500
```

---

# ⚙️ Backend Configuration

Update the API URL inside `script.js`.

### Local Development

```javascript
const API_URL = "http://localhost:8080";
```

### Production

```javascript
const API_URL = "https://your-backend-url.com";
```
